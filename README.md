# Quantum Computing FDP Website

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A modern, responsive website for the AICTE Sponsored ATAL FDP on Quantum Computing - "BITS TO QUBITS" organized by the Department of Artificial Intelligence and Machine Learning at HKBK College of Engineering.

## 🚀 Features

- **Responsive Design**: Works on all device sizes
- **Modern UI**: Built with React, TypeScript, and Tailwind CSS
- **Interactive Components**: 
  - Schedule/Agenda
  - Resource Persons section
  - Organizing Committee
  - Venue & Contact Information
  - Google Maps Integration

## 🛠️ Tech Stack

- **Frontend**: 
  - React 18
  - TypeScript
  - Tailwind CSS
  - Vite
  - Lucide Icons

## 🎨 Design

- **Color Scheme**:
  - Quantum Navy
  - Quantum Cyan
  - Dark theme with vibrant accents
- **Typography**: Modern, clean fonts with excellent readability
- **Animations**: Subtle hover effects and transitions
